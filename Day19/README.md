## Day 19 - Instances, State, and Higher Order Functions

**Stage:** Intermediate

**Project:** Turtle Race (**try [here](https://replit.com/@GloryOdeyemi/Turtle-Race?v=1)**).

**Topics:**
* Instances
  ```
  turtle1 = Turtle()
  turtle2 = Turtle()
  ```
* State
  ```
  turtle1.color("green")
  turtle2.color("blue")
  ```
* Higher Order Function - function that can work with another function
  ```
  def a(n):
    do something

  def b(num):
    a(num)
  ```
