## Day 13 - Debugging: How to Find and Fix Errors in your code.

**Stage:** Beginner

**Debugging Tips**
1. Describe the problem
2. Reproduce the bug
3. Play computer
4. Fix the errors
5. Print is your friend
6. Use a debugger
7. Take a break
8. Ask a friend
9. Run the code often
10. Ask StackOverflow
